OXU QRVIEW — trang test đã chuyển vào app

Trang test đầy đủ (nhập STK/số tiền → VietQR API → chuỗi EMVCo → gửi COM) nằm trong SPA:

Mở Test QR OXU trong app

Hoặc từ menu: Test QR OXU (route #qr-oxu-test).

Tài liệu lệnh COM (115200 baud)

  • JUMP(1); — màn hình QR
  • QBAR(0,<payload>); — hiển thị QR (link hoặc chuỗi VietQR EMVCo)
  • SET_TXT(0,AGRIBANK); — tên ngân hàng
  • SET_TXT(1,STK: 84593827363); — số tài khoản
  • SET_TXT(2,230.000); — số tiền hiển thị
  • BL(20); — độ sáng 0–255

Mỗi lệnh kết thúc bằng \r\n khi gửi qua Web Serial.

VietQR payload: đăng ký API tại my.vietqr.io, gọi POST /v2/generate, lấy data.qrCode thay cho oxu.vn trong QBAR(0,...).
